ACQUIRED_R

Field ACQUIRED reader - Specifies if the lock is acquired. This field is set to ‘1’, if a ACQUIRE read transfer successfully acquires the lock (the ACQUIRE read transfer returns ACQUIRE.SUCCESS as ‘1’). If zero, P, NS, PC, and MS are not valid.
