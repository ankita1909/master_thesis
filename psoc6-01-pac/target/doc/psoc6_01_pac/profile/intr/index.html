CNT_OVFLW_R

Field CNT_OVFLW reader - This interrupt cause field is activated (HW sets the field to ‘1’) when an profiling counter overflow (from 0xFFFFFFFF to 0x00000000) is captured. There is one bit per profling counter. SW writes a ‘1’ to a bit of this field to clear this bit to ‘0’ (writing 0xFFFFFFFF clears all interrupt causes to ‘0’).

CNT_OVFLW_W

Field CNT_OVFLW writer - This interrupt cause field is activated (HW sets the field to ‘1’) when an profiling counter overflow (from 0xFFFFFFFF to 0x00000000) is captured. There is one bit per profling counter. SW writes a ‘1’ to a bit of this field to clear this bit to ‘0’ (writing 0xFFFFFFFF clears all interrupt causes to ‘0’).
