Expand description

The mode controls how the USB SIE responds to traffic and how the USB SIE changes the mode of that endpoint as a result of host packets to the endpoint.

Value on reset: 0

Variants

DISABLE

0: Ignore all USB traffic to this endpoint

NAK_INOUT

1: SETUP: Accept IN: NAK OUT: NAK

STATUS_OUT_ONLY

2: SETUP: Accept IN: STALL OUT: ACK 0B tokens, NAK others

STALL_INOUT

3: SETUP: Accept IN: STALL OUT: STALL

ISO_OUT

5: SETUP: Ignore IN: Ignore OUT: Accept Isochronous OUT token

STATUS_IN_ONLY

6: SETUP: Accept IN: Respond with 0B data OUT: Stall

ISO_IN

7: SETUP: Ignore IN: Accept Isochronous IN token OUT: Ignore

NAK_OUT

8: SETUP: Ignore IN: Ignore OUT: NAK

ACK_OUT

9: SETUP: Ignore IN: Ignore OUT: Accept data and ACK if STALL=0, STALL otherwise.
Change to MODE=8 after one succesfull OUT token.

ACK_OUT_STATUS_IN

11: SETUP: Accept IN: Respond with 0B data OUT: Accept data

NAK_IN

12: SETUP: Ignore IN: NAK OUT: Ignore

ACK_IN

13: SETUP: Ignore IN: Respond to IN with data if STALL=0, STALL otherwise OUT: Ignore

ACK_IN_STATUS_OUT

15: SETUP: Accept IN: Respond to IN with data OUT: ACK 0B tokens, NAK others
