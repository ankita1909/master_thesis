CC_MATCH_MODE_R

Field CC_MATCH_MODE reader - Determines the effect of a compare match event (COUNTER equals CC register) on the ‘line_out’ output signals. Note that INVERT is especially useful for center aligned pulse width modulation. To generate a duty cycle of 0 percent, the counter CC register should be set to ‘0’. For a 100 percent duty cycle, the counter CC register should be set to larger than the counter PERIOD register.
