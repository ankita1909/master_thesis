STOP_SEL_R

Field STOP_SEL reader - Selects one of the 16 input triggers as a stop trigger. In PWM, PWM_DT and PWM_PR modes, this is the kill trigger. In these modes, the kill trigger is used to either temporarily block the PWM outputs (PWM_STOP_ON_KILL is ‘0’) or stop the functionality (PWM_STOP_ON_KILL is ‘1’). For the PWM and PWM_DT modes, the blocking of the output signals can be asynchronous (STOP_EDGE should be NO_EDGE_DET) in which case the blocking is as long as the trigger is ‘1’ or synchronous (STOP_EDGE should be RISING_EDGE) in which case it extends till the next terminal count event.

STOP_SEL_W

Field STOP_SEL writer - Selects one of the 16 input triggers as a stop trigger. In PWM, PWM_DT and PWM_PR modes, this is the kill trigger.
