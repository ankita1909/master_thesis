CA_EN_R

Field CA_EN reader - Cache enable: 0: Disabled. The cache tag valid bits are reset to ’0’s and the cache LRU information is set to ’1’s (making way 0 the LRU way and way 3 the MRU way). 1: Enabled.

CA_EN_W

Field CA_EN writer - Cache enable: 0: Disabled. The cache tag valid bits are reset to ’0’s and the cache LRU information is set to ’1’s (making way 0 the LRU way and way 3 the MRU way). 1: Enabled.
