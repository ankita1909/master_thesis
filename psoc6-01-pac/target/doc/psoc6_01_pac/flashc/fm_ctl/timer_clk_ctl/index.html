TIMER_CLOCK_FREQ_R

Field TIMER_CLOCK_FREQ reader - Clk_t frequency divider to provide the 1MHz reference clock for the Regif Timer. Equal to the frequency in MHz of the timer clock ‘clk_t’. Example: if ‘clk_t’ has a frequency of 4 MHz then this field value is ‘4’ Max clk_t frequency = 100MHz. This field is updated at runtime with the ’SW_TIMER_CLOCK_FREQ ’ value from the HV parameters table
